# Security Policy

## Reporting a Vulnerability

Please report security issues privately. Do not open a public issue.

- Email: security@parsezero.dev
- Include: description, repro steps, affected versions, environment, and any sample CSV if relevant.

We will acknowledge receipt within 2 business days and provide a timeline for investigation and remediation.

## Supported Versions

We support the latest released NuGet package and the current `main` branch. Older versions may not receive patches.

## Coordinated Disclosure

If a vulnerability is confirmed, we will:
- Prepare and validate a fix
- Publish a patched release to NuGet
- Acknowledge reporters (if desired)
- Coordinate public disclosure timing when needed
